# 開発環境
- C++
- MinGW + CMake

# ライブラリ
できる限りピュアなC++を利用するつもりですが、ライブラリを利用する可能性があります。その際はこのリポジトリにてお知らせします。

# 貢献方法
このプロジェクトへの貢献方法は3つあります。

1. あなたの経験を活かしてこのプロジェクトのリファクタリングを行い、プルリクエストを作成すること
    - これは、必ず`dev`ブランチから新しいブランチを作成した後に、作業してください
    - PRは必ず`dev`に向けて行ってください
2. あなたの知識を活かしてこのプロジェクトに対して、機能要求イシューを作成すること
    - これは、必ずテンプレートに沿って作成してください
    - 既存、代替可能性のあるものは対応しません
3. 利用していて不具合を見つけたことを報告(Issue作成)すること
    - 小さいミスでも歓迎します
    - ただし、あなたのミスの可能性もあるのでよく調査してから作成してください
